• Role: staff (เบิก) / manager (อนุมัติ+รับเข้า) / admin (จัดการระบบ)
• Locations: คลังที่เข้าถึงได้ — ปล่อยว่าง = Super (เห็นทั้งหมด)
ชื่อ แผนก Role 🏢 Locations สถานะ เบิก
🏢 จัดการ Location

เลือก location ที่ user นี้เข้าถึงได้ — ไม่เลือกอะไรเลย = Super (เห็นทุก location)
